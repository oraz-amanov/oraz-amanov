## Projects

### 1. Digital Twin of a Drilling Rig (in progress)
- Building a simplified digital representation of a drilling system and its components
- Implementing a Random Forest regression model to predict Rate of Penetration (ROP) from parameters such as Weight on Bit (WOB) and RPM
- Exploring how system-level variables influence drilling performance

Key insight:  
Initial results show limited prediction accuracy, highlighting challenges in capturing complex drilling dynamics using purely data-driven models.

---

### 2. Drill Bit Penetration Simulation & Optimization (in progress)
- Developing a simulation of drill bit-rock interaction
- Integrating a regression model for ROP prediction
- Preparing the system for integration with a Reinforcement Learning algorithm (developed by a collaborator) for autonomous drilling optimization

Key insight:  
This project explores the transition from prediction to control, where model accuracy directly affects optimization performance. This project will further be integrated with project 1 to enhance the digital twin system.

---

### 3. Lab-Based Digital Twin for Drill Bit Penetration (planned)
- Reproducing a laboratory drilling experiment as a digital model
- Integrating real experimental data into the simulation
- Aiming to bridge the gap between theoretical models and physical systems

Goal:  
To create a more reliable and validated simulation environment for studying drilling behavior.
